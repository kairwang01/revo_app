# Changelog

## 0.2.0 - Curated marketplace polish
- Rebuilt every page module with curated listings, PDP trade-in flow, checkout form, and order tracking.
- Added responsive tokens, refined base + component styling, and enhanced accessibility helpers.
- Implemented header city selector, sticky tab bar with SVG icons, and FAB interactions.
- Expanded mock data with pricing, ratings, availability, and order metadata for Apple/Samsung devices.
- Strengthened cart & checkout logic with city tax, quantity steppers, payment toggles, and analytics hooks.
- Upgraded PWA shell, router hooks, REM scaling utility, and service worker asset manifest.

- Scaffolded SPA shell with hash router and dynamic imports
- Implemented responsive REM scaling and safe-area CSS variables
- Added PWA manifest and Service Worker (cache-first static, network-first API)
- Built header with city selector, search, and cart badge
- Built sticky bottom tab bar with centered FAB "Trade-In"
- Implemented core pages: Home, Curated, Products, Product Detail, Account, Login, Register, Settings, Cart, Checkout, Order Tracking, Not Found
- Wired mock API + localStorage for auth, cart, city, wallet
- Added accessibility improvements and performance-friendly defaults
